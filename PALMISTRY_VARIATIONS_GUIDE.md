# Система Вариаций для Хиромантии

## 🎯 Проблема

Ранее приложение использовало только **3 варианта** для каждой области жизни (любовь, карьера, судьба):

```
3 варианта × 3 области = 9 вариаций всего
```

**Результат:** Если семья членов сканирует ладони в один день, они видят идентичные или почти идентичные предсказания.

## ✅ Решение

### Уровень 1: Расширенные пулы (текущая реализация)

Каждая область теперь имеет **30 вариаций**:

```
30 вариантов × 3 области = 90 различных вариаций
```

**Файлы:**
- `lib/palmistry-readings.js` — 30 вариаций для каждой области
- `lib/palmistry-generator.js` — обновлен для использования новых пулов

**Как работает:**
```javascript
const { generateReading } = require('./lib/palmistry-generator');

// Каждый sessionId порождает разный текст (благодаря seeded random)
const session1 = generateReading({ sessionId: 'user-123-001', name: 'Анна' });
const session2 = generateReading({ sessionId: 'user-456-001', name: 'Мария' });

// session1.full ≠ session2.full (разные вариации из пула 30)
```

---

### Уровень 2: Компонентный подход (опционально)

Для еще большего разнообразия можно разбить каждый текст на компоненты:

```
3 введения × 3 инсайта × 3 завершения = 27 комбинаций
27 комбинаций × 30 основных вариаций = 810+ различных текстов
```

**Файл:** `lib/palmistry-reading-mixer.js`

**Использование:**
```javascript
const mixer = require('./lib/palmistry-reading-mixer');

function seededRandom(seed) {
  let s = 0;
  for (let i = 0; i < seed.length; i++) {
    s = (s * 31 + seed.charCodeAt(i)) >>> 0;
  }
  return function () {
    s ^= s << 13;
    s ^= s >> 17;
    s ^= s << 5;
    return (s >>> 0) / 0xFFFFFFFF;
  };
}

const rng = seededRandom('session-id');
const customLoveText = mixer.generateCustomLoveText(rng);
```

## 📊 Варианты по Барнум-эффекту

Каждый из 30 вариантов включает 2–4 архетипа:

### Любовь (LOVE_READINGS)
- ✅ Независимость + близкие отношения
- ✅ Глубина чувств + сдержанность в выражении
- ✅ Ранимость как дар
- ✅ Поиск подлинности
- ✅ Баланс между отдачей и принятием

**Пример (вариация #1):**
> "Линия сердца в вашей ладони говорит о способности любить редко, но глубоко. Вы чувствуете сильнее, чем говорите об этом. Если прошлые отношения оставили раны — ладонь показывает, что этот цикл завершается. Впереди партнёрство, где вас увидят и оценят полностью."

### Карьера (CAREER_READINGS)
- ✅ Нереализованный потенциал
- ✅ Видение сути как преимущество
- ✅ Значимая работа > деньги
- ✅ Лидерство (тихое, но реальное)
- ✅ Переход на следующий уровень

**Пример (вариация #1):**
> "Линия головы и линия судьбы в вашей ладони показывают редкое сочетание талантов, которые пока не раскрыты полностью. Вы способны на большее, чем текущее положение. Скоро появится возможность — важно не пропустить."

### Судьба (DESTINY_READINGS)
- ✅ Трансформация через испытания
- ✅ Нелинейный путь, полный смысла
- ✅ Влияние, которое ещё не достигло пика
- ✅ Циклы роста
- ✅ Наследие, которое переживает смерть

**Пример (вариация #1):**
> "Общий рисунок вашей ладони говорит о жизни с несколькими ключевыми точками трансформации. Вы уже прошли через них — и каждый раз выходили сильнее. Впереди — ещё одна. Но в отличие от прошлых, эта будет переходом из готовности, а не из боли."

## 🔧 Техническая реализация

### Текущее состояние (palmistry-generator.js)

```javascript
const { LOVE_READINGS, CAREER_READINGS, DESTINY_READINGS } =
  require('./palmistry-readings');

function generateReading({ sessionId, name, gender, focusArea, handScore = 0.7 }) {
  const rng = seededRandom(sessionId || 'default');

  // ... другой код ...

  // Теперь использует 30 вариаций вместо 3
  const loveText    = pick(LOVE_READINGS,    rng);    // 1 из 30
  const careerText  = pick(CAREER_READINGS,  rng);    // 1 из 30
  const destinyText = pick(DESTINY_READINGS, rng);    // 1 из 30

  // Результат: полный текст содержит 1 из 27000 комбинаций
  // (хотя обычно выбирается одна из трех областей, в зависимости от focusArea)
}
```

### Альтернатива: Пользовательское сидирование (User-based consistency)

Если хотите, чтобы один пользователь всегда видел одинаковую вариацию (но разные пользователи видели разные):

```javascript
const crypto = require('crypto');

function getReadingByUser(userId) {
  const hash = crypto.createHash('sha256').update(userId).digest('hex');
  const index = parseInt(hash.slice(0, 8), 16) % LOVE_READINGS.length;
  return LOVE_READINGS[index]; // Всегда одна и та же вариация для этого юзера
}

// Но сейчас лучше использовать sessionId, чтобы каждый скан давал свежий текст
```

## ✨ Психологические архетипы (Барнум-эффект)

Каждый текст сбалансирован, чтобы быть:

1. **Универсально релевантным** — "вы часто сомневаетесь" (почти все так делают)
2. **Личностно значимым** — использует имя, гендер, историю
3. **Позитивным и эмпаурирующим** — "впереди лучшее"
4. **Не предсказывающим** — избегает "вы встретите человека 15 августа"
5. **Интроспективным** — фокусируется на чувствах, не на фактах
